# Jekyll

WITH DEVKIT - Ruby+DevKit 2.7.5-1 (x64) 를 클릭 <br>

cmd 명령어. <br>

gem install jekyll <br>

[jekyll 명령어 설명](https://jekyllrb.com/docs/){:target="_blank"}

```bash
gem install bundler
gem install bundler jekyll

ruby -v
jekyll -v

bundler exec jekyll serve

bundle install
bundle update
```
